# Analysis Methods Reference

## Summary Statistics
- Mean: Average value, sensitive to outliers
- Median: 50th percentile, robust to outliers
- Std Dev: Spread of data around the mean
- IQR: Q3 - Q1, measures middle 50% spread

## Correlation Analysis
- Pearson: Linear relationship, assumes normal distribution
- Spearman: Monotonic relationship, non-parametric
- Thresholds: |r| > 0.7 strong, 0.3-0.7 moderate, < 0.3 weak

## Statistical Tests
- T-test: Compare means of two groups (independent or paired)
- ANOVA: Compare means of 3+ groups
- Chi-square: Test independence of categorical variables
- Shapiro-Wilk: Test for normality

## Visualization Guidelines
- Histogram: Distribution of a single variable
- Box plot: Distribution with quartiles and outliers
- Scatter plot: Relationship between two variables
- Heatmap: Correlation matrix visualization
- Bar chart: Categorical comparisons
